실행하는 Run.exe 파일은 동 폴더 내의 Run.py 소스코드를 pyinstaller로 생성한 것입니다.
exe파일을 믿지 못하시는 분들은 파이썬을 설치해 Run.py 코드를 직접 실행하시면 됩니다.


<준비물>

마이크로소프트 엑셀
구글 크롬



<사용법>

Run.exe
msedgedriver.exe
base.xlsm
3개의 파일은 반드시 같은 공간에 있어야 합니다.



1. base.xlsm 에서 기본 데이터 입력

base.xlsm 파일을 열어서 검색시트로 갑니다. (시트는 엑셀 하단에서 선택 가능하며, 총 3개의 시트가 있습니다.)
시트 우측에 로그인 정보를 입력합니다. 이 정보는 오직 로그인에만 사용됩니다.
다시 말하지만 이 프로젝트는 오픈소스입니다. 만약 이 파일을 깃허브가 아닌 다른 곳에서 받았다면 해킹의 위험이 있습니다.

로그인 정보 밑에 대성공확률, 제작비용감소율, 활동력감소율을 입력하는 칸이 있습니다.
본인 영지에 맞게 '숫자만' 입력해주시면 됩니다.
ex) 5.25 라고 입력하면 자동으로 뒤에 '%'가 붙습니다.

코스튬 구입 등으로 인해 확률이 변경되면 base.xlsm 파일에서 저 부분만 수정해주시면 됩니다.

!!! 이 작업 외에는 절대 base.xlsm을 수정하지도 말고 삭제하지도 마세요.


2. Run.exe 실행

프로그램이 자동으로 로그인 후 거래소에서 현재 최저가를 쭉 탐색합니다.
이 작업은 약 15~20초 걸립니다.
작업이 끝나면 실행한 시각을 파일 이름으로 새 엑셀 파일을 폴더 내에 생성합니다.
ex) 0906_010046.xlsm (9월 6일 1시 00분 46초라는 뜻)


3. 생성된 엑셀 파일을 열어 확인합니다.

엑셀을 열 때 위 메뉴쪽에 '콘텐츠 사용'을 눌러주세요.

기본 사용법
'검색' 시트에서 하단의 아이템명 리스트에서 원하는 아이템을 클릭하세요.
해당 아이템의 거래소가, 제작비용, 최저가-1골기준 활동력1만당 순이익을 보여줍니다.
아이템 리스트 바로 위에 1~3위는 활동력 1만당 순이익을 나타냅니다. (최저가-1골 판매기준)
참고로 1~3위에 나타난 아이템이름을 클릭해도 됩니다.



제작을 할 때마다 Run.exe 를 실행하여 그 때 시세에 맞는 데이터로 비교하시면 됩니다.
base.xlsm을 제외한 나머지 생성된 엑셀파일은 삭제해도 됩니다.

사용 중 에러가 발생하면 껐다가 다시 실행해보시고
